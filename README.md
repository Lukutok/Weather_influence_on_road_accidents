# Влияние погодных условий на ДТП
**Цель**: собрать и структурировать в базе данных сведения о ДТП и погодных условиях. Проанализировать взаимосвязь между параметрами погоды и частотой ДТП (степенью тяжести).

**География данных**: г. Екатеринбург, г. Пермь.

## Сбор данных
Первым этапом собираем через Open-Meteo API (Historical Weather API) погодные почасовые данные для выбранных городов, начиная с 2015 года и по сегодняшний день.
